# Clinical Data Analysis – Factors Associated with Diabetes Progression

## Overview
This project explores clinical variables associated with diabetes disease progression using statistical analysis and machine learning techniques. The goal is to identify meaningful predictors and demonstrate data-driven clinical interpretation.

## Objectives
- Identify key predictors of diabetes progression
- Perform exploratory data analysis (EDA)
- Apply machine learning models for predictive analysis
- Translate analytical findings into clinically relevant insights

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Key Findings
- BMI demonstrated the strongest association with disease progression
- Blood pressure showed moderate predictive influence
- Several clinical variables contributed to model performance
- Data-driven analysis may support clinical decision-making and risk assessment
